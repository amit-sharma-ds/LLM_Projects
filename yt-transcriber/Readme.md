# AI YouTube Transcript → Notes Converter

An LLM-powered tool that converts YouTube video transcripts into clean, structured notes.  
Just paste a YouTube link and get summarized learning instantly.

## Features
- Extracts YouTube transcripts
- Converts into structured notes
- Fast Streamlit UI
- Works with subtitle-enabled videos

## Tech Stack
- Python
- Streamlit
- Gemini API (LLM)

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
