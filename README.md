# LLM_Projects

# AI / Data Science Projects Repository

This repository contains multiple AI, NLP, and data science projects built using Python and LLM-based techniques. Each project focuses on solving real-world problems using modern machine learning and natural language processing approaches.

---

## Data Science Project

A data science and machine learning project repository focused on experimenting with real-world datasets, data preprocessing, and model building.  
It includes exploratory data analysis, feature engineering, and evaluation of ML workflows using Python.

---

## PDF_Summarizer

An LLM-powered tool that extracts text from PDF files and generates concise summaries.  
It helps in quickly understanding long documents by converting them into key insights using natural language processing techniques.

---

## yt-transcriber

An AI-based YouTube transcription tool that converts video audio into accurate text transcripts.  
It enables easy note-making, summarization, and improves accessibility of video content using speech-to-text and NLP.
